Regression

Most generally expressed, regression is the analysis of the relationship between two variables, specifically of the change in the dependent variable as the independent variable (or variables) changes.

If we can create robust models then we can reason about the quantitative relationships between phenomena with some precision and, in a more applied sense, we use historic data for forecasting, for making reasoned predictions from the observed to the unobserved.
