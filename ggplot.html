Using ggplot2

qplot

ggplot2 is a package you can add to R that provides sophisticated plotting functions.  It follows a scheme for visualisation of data called grammar of graphics that was developed by Leland Wilkinson to describe the components and layout of a plot or graph.

First, have a look at this video that describes the very basic use of the qplot function that is part of ggplot2. 

When  you have watched the video, then work through the section on qplot on this Quick-R blog page.

ggplot

You will see on the Quick-R blog page that after introducing the quick plots, the second section is an example ofcustomising  graphs using the ggplot function from the ggplot2 package.
