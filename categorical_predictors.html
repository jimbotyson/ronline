Categorical predictors

The simple linear regression model and the multiple inear regression model introduced in previous sections can be augmented by the introduction of categorical predictor variables.

The coefficient of a continuous predictor - for example score in maths - is understood as the slope of he least squares best fit line and in the line for the equation appears as multiplier of the independent variable.

The categorical predictors are interpreted slightly differently.  We will imagine using dummy coding to create our model.  In dummy coding we take a variable - let us use gender as an example - and recode it as a new variable female with values 0 for a male subject and 1 for a female subject.  The coefficient calculated for this, then, represents the difference in scores between subjects with a zero score and those with a score of one.  The same logic holds for variables with more than two levels.

Looking at the relationship between class membership and examination performance in the results data, we ask what is the effect of member ship of class two or three against class one in this data on avxm?

First, you will need to ensure that your variable can be understood by R as a factor and you can test this with is.factor(variable) and if necessary you can use as.factor(variable) to either create a new factor variable based on the existing one or to coerce the interpretation of the variable as a factor.

The model is created with lm() and the categorical variable is introduced in the usual way, introduced by + if there are multiple independent variables.

The lowest vaule of your factor is the default reference level - the other levels effect is interpreted relative to this.  You can force R to set any level to be the reference level using the function C().  This function allows you to determine which of the levels of the factor is the base.

Create a regression model to illustrate the effect of class on first maths and then on avxm.
