ANOVA

ANOVA is used to compare differences of means among more than 2 groups.  It determines whether the means are different by comparing the variability within the groups (error) to the variability between groups (the treatment effect: the effect of the independent variable or factor).  That's why it's the analysis of variance rather than the analysis of means.

The null hypothesis for the ANOVA is that the mean value of the variable of interest does not differ between groups.

The alternative hypothesis is that at least one mean is different.

The main statistic computed by the procedure is the F value.  This is the ratio of the between group variation and the within group variation.  If these quantities are close to equal then the F statistic is close to one.

R provides a direct test for a one way anova (ie with a single factor) and provides for more complex models using either the function aov() or the linear model lm().  Since the aov() function will cover both one way and more complex models, it is sensible to use it for both cases.

Before carrying out the ANOVA test it is useful to check the descriptive statistics and to visualise the variables of interest with box plots.

The instructions for carrying out the tests are on this Quick-R Blog. 

There is a detailed Lynda tutorial for a one way case.  And another for the two factor ANOVA.

Performing a one-way ANOVA

First, read the results data into a dataframe.  The data are in the file results.csv on the website http://www.ucl.ac.uk/~ccaajim.

Next, generate a new variable avxm which holds the average of the three exam scores maths, english, history.

Explore the avxm variable split by class and create box plots of avxm split by class.

Check the variances of the avxm scores by class.

Carryout an ANOVA to compare the mean avxm scores by class.  If the result of the ANOVA indicates significant differences, carry out Tukey's Honest Significant Differences test to find out where the pairwise differences lie.

by(avxm,class,summary)

boxplot(avxm~class)

bartlett.test(avxm,class)

model<-aov(avxm~class)

summary(model)

TukeyHSD(model)
