Basics of plotting

Simple R plots

We have already seen a simple histograph created with hist(). The code below illustrates how we can modify a graph incrementally by adding elements. It uses the built in data set mtcars and create a histogram of the values in the mpg column then overlay a curve with a normal distribution with mean equal to the mean value of mpg and standard deviation equal to its standard deviation:

> hist(mtcars$mpg,prob=TRUE)
> curve(dnorm(x,mean=mean(mtcars$mpg),sd=sd(mtcars$mpg)),add = TRUE)

This graph is created in two steps. The first line produces this graph

The second overlays the curve producing this

You will see that this same process of plot elaboration holds for other graph types.

Graphing Exercise

The data in the file results.csv below contains exam scores for thirty school students: fifteen boys and fifteen girls.  They have been scored for exams in maths, english, and history out of 100.

For each student, gender is coded 1=girl and 2=boy.  The class variable tells us which class a student is in and has levels 1, 2, or 3.

We would like to see how scores vary for each exam and for the exam score average by gender and class.  To do this you are asked to first generate the variable avxm which should hold the average of the three exam scores for each pupil.  Then,  produce the boxplots: for each score you will need to create two box plots split by gender and three split by class.

The graphs need to be laid out so as to make the comparison easy.

You can generate avxm using the assignment operator <- and some arithmetic.

To create the plots you will need to use boxplot.  The simplest boxplot command requires a continuous variable and a factor.  The syntax is

> boxplot(score~factor)

For example

> boxplot(avxm~class)

creates three box plots for the avxm variable split on the class factor.

You can also use a subset option

> boxplot(avxm~class,subset = gender=="1")

> boxplot(avxm~class,subset = gender=="2")

You should test these lines of code to be sure that you understand them.  Incidentally they reveal something about the distribution of gender over class.  What is it?

The par function allows you to create parameters for use in a plot.  One parameter allows you to specify that a sequence of plots command will be layed out as sub-plots in columns and rows.  You can specify the number columns and rows with the multiple figure options.  They are

mfrows=c(nr nc) plot the graphs row-wise
mfcols=c(nr nc) plot the graphs column-wise

So the command

par(mfrow=c(3,2))

Will layout any graphing commands that follow in a grid of three rows and two columns, filling across the rows first.  The code

par(mfcol=c(3,2))

Will layout any graphing commands that follow in a grid of three rows and two columns, but filling down the columns first.

Using these commands, create a suitable set of plots, conveniently laid out to illustrate the spread of exam scores across class and gender.

For class==2 you should create a graph of avxm by gender.  To do this you will need to use the subset parameter on the boxplot command.  Use R help or any other resources you can find to research this.

Feedback

There is no single correct solution but this code should give you some idea of how to apporach this

> par(mfrow=c(1,3))

> boxplot(maths~class)

> boxplot(english~class)

> boxplot(history~class)

To create the plot of avxm split on gender for class two only:

boxplot(avxm~gender, subset = class==2)
