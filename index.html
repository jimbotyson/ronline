R: a first course

Objectives

After completing this course you should be

  • Confident with R Studio and basic R commands
  • Able to import, explore and manipulate your data in R
  • Create and modify simple R plots
  • Analyse data using some hypothesis tests
  • Create and evaluate regression models involving both continuous and categorical variables
  • Know how to get help and find documentation

Are you ready to start?

Pre-requisite knowledge

This course assumes that you have some knowledge and skills before you start.  For example, it assumes that you know some statistics:

  • descriptive statistics such as measures of
    • location
    • difussion
    • shape
  • the use of graphs and plots
  • the different kinds of data that are analysed statistically such as
    • continuous versus discrete data
    • ordinal versus ration data (and the other levels of measurement)
  • an understanding of hypothesis testing, significance and confidence intervals

Digitial Capabilities

The course also assumes that you have basic computing skills such as file management and using a text editor or word processing program.

Learning online

The course also assumes that you are comfortable with online learning.  What previous experience of online learning do you have?  How did you find any previous online learning experiences?  Did you enjoy them?  Did you find them worthwhile?

Ready to go?

Take a moment to think about the objectives of the course and some of the assumptions outlined above.  Do you understand the objectives and do they correspond to what you want from the course? Do you have the kind of pre-knowledge that the course assumes?
