Basic command line statements

Creating simple data objects

You should open RStudio on your computer and try out the commands in this section as you go.  You can copy the actual code if you want to avoid a lot of typing.

The console when it is ready for you to enter a command shows a prompt:

>

For our first R command we will enter a bit of simple arithmetic:

> 2+2

[1] 4

>

Or these two lines

sum(2,3,4,5,6,7,8,9,1,10)

sum(2,3,4,5,6,7,8,9,1,10)/10

R acts as a simple calculator here and it returns the result of its calculation to the screen for us.

This page has a list of useful R language functions. You will find it very useful when you start on the exercises.

But what is the [1] telling us?

When R displays data for us, it will wrap the data lines to fit your window width. The numer in square brackets at the beginning of each line tells you with which element that line begins (the first, tenth, etc). It is entirely cosmetic.

You can see this by entering the following command:

> x<-c(1:100)

> x

which displays

R uses the symbol <- to take values on the right hand side and assign them to variables on the left hand side.

The symbol <- is read as gets.

In this case the right hand side is

c(1:100)

Which means combine into a vector the numbers from 1 to 100. Often the c() function is read as collect.

A vector is a collection of elements of the same basic type (for example of numbers or of characters of the alphabet). Vectors are fundamental to the way R works.

Because R understands vectors we can easily operate on all the elements of our collected values. For example

> y<-x^2

> y

squares all the elements of x. (You notice that we use ^ for raising to a power).

Of course, you may not wish to create all your data objects by typing in values at the command line. Next we will see how R provides ways for you to read data from external sources - files created in other applications such as, for example, Microsoft Excel.

We are going to look at importing data without worrying for now about the type of object it is imported into. Later on we will look at different kinds of R data object.

Simple data selection

Selecting by index

The elements of a vector each have an index: that is their position in the vector.  In R the indices start at 1 and continue for the length of the vector.  So

x[1]

Produces

[1] 1

and

x[35]

Produces

[1] 35

Using the logical vector

Logical vectors can be used in interesting ways.

Some values of x are even: divided by two the remainder is 0.

We can find those values using the modulo operator: %%.

> x%%2==0

This is the result

  [1] FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE
 [13] FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE
 [25] FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE
 [37] FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE
 [49] FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE
 [61] FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE
 [73] FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE
 [85] FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE FALSE  TRUE
 [97] FALSE  TRUE FALSE  TRUE

Now, recall that y contains the squares of x.  So, we can find the square of x when x is even like this:

> y[x%%2==0]

or we can do this

z<-x%%2==0

And here z will be the logical vector above.  Now we can select the y values where x is even like this:

y[z]
