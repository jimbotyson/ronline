Help

R help and documentation

There is a comprehensive on-line introduction to the language on the CRAN pages.

Help in RStudio

In RStudio you can use the function help() to access help in various ways.

You should try

help.start()

to see the online manuals for the language.

You can use the command with the name of a function for specific help, for example

help(lm)

Which finds the help page for the function _lm()_ - the function that fits linear models. You can also search by keyword - for instance _categorical_ - using the following syntax:

??categorical

Later you will learn how to install add-on packages for R and you will be able to get help for packages in the same way.
