Summary statistics

Summary statistics

R has functions for some descriptive statistics, for example mean(), and sd(), min(), max(), range and quantile().

These are used quite straightforwardly on single vectors. Let us use the built-in cars data set.

>cars
>head(cars)
  speed dist
1     4    2
2     4   10
3     7    4
4     7   22
5     8   16
6     9   10

> mean(cars$dist)
[1] 42.98
> sd(cars$speed)
[1] 5.287644

A useful tweak to obtaining descriptive values in this way involves the funciton sapply(). This function takes a data object as its first argument and a function as its second and applies the function to all data vectors in the object. So

sapply(cars,mean)
speed  dist 
15.40 42.98

One thing to be aware of when using sapply() (and other functions that make numeric computations) is that missing data in R are represented by NA and the attempt to compute a numeric function will return an error because of these cases. Let us consider this vector and the attempt to compute its mean:

> x<-c(1,NA,2,3)
> mean(x)
[1] NA

to avoid this, we will specify that R should remove the cases (known as listwise deletion) with missing data.

> mean(x,na.rm=TRUE)
[1] 2

The expression na.rm=TRUE tells the R interpreter to remove the missing cases before making the calculation.
