First task

Instructions

You should now attempt the first task.

The file ss13hca.csv contains a single year subset of data from the American Community Survey Public Use Micro-data Sample which reports on characteristics of households in California.

You will need the data dictionary in file PUMSDataDicttionarysingleyear2013.txt. This file will explain what data the columns in this file contain.

You will need to find the variables that you need to examine for this task from the data dictionary.  You will probably want to use the find function in your text editor to search this data dictionary.

Read the data file into an R data frame.

From the complete data frame create a new data frame containing data meeting only these constraints:

  1. Only one adult in the household is in the labour force.
  2. Only include Housing units.
  3. Only include households with a dwelling on a less than one acre plot.
  4. Only include households with children.
  5. Only include households with family income between $75,000 and $150,000
  6. Exclude households where a grandparent is head of household and no parent is present.
  7. Exclude households with no laptop.

Choose a suitable name for this data frame.

How many cases are there that meet these conditions?

Create a vector of the income of those households where no child is under six years old.
