Testing for equality of variance

The comand

var.test(sample1,sampel2

will test two samples with an F test. The null hypothesis is that the variances are equal.

Indipendent t-test activity

For this activity you will read a file in Stata format remotely (it is on a website) into a dataframe and then carry out an independent samples t-test.

The data are from schools in California and the variables of interest are api00 and yr_rnd.  Some schools in the United States run an all year program - there is no long summer break but (usually) four ten week blocks seperated by one or two week breaks.  Some people think that this improves school performance.  And, naturally some disagree.

The api00 variable is a measure of school performance and the yr_rnd variable indicates whether the school ran an all year round program of teaching. 

On the data supplied, is school performance the same for year round and non year round schools?

To read the data you need the following code:

install.packages("foreign")

require("foreign")

myapi<-read.dta("http://www.ucl.ac.uk/~ccaajim/apilog.dta", warn.missing.labels = FALSE)

(This installs a package that allows you to read some proprietary data formats including Stata data, loads it and then reads a file.  The warn.missing.labels option suppresses warnings that there are labels in the data that are not coded as specified in the file, which in this case is non an issue).

To help you start, here is the code for the test of variance.

var.test(api00[yr_rnd=="yrrnd"],api00[yr_rnd=="not_yrrnd"])

What answer can you give to the question asked above about school performance.
