Add on packages

Since R is a programming language, you can do with it anything a computer can be programmed to do. However, there is little point in all of us writing the code for a wheel.

Members of the R community contribute to the R project by creating packages which provide extended functionality or package useful functionality together in a way that is convenient for the R user.

The R documentation project pages have a list of packages organised by topic which you can usefully browse to see what is available.

To download and install a package use the command

install(packagename)

Having installed the package you make it available with the command

library(packagename)

You will also see the command require used to load packages, but this is not recommended. If you would like to know why you can read this blog post.
