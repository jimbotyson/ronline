Third task

Two useful descriptive measures on continuous data are included in the package moments. After installing and loading the package you can compute the skewness() and kurtosis() of data.

Supplementary reading

You will find an introduction to the shape parameters in this Wikipedia article.

Instructions

First, create some data with a markedly high kurtosis (this is a rather crude way to do that but the alternative would mean manipulating probability distribution functions and life's too short)

a <- rnorm(5000,0,2)

b <- rnorm(1000,-2,4)

c <- rnomr(3000, 4,4)

kurtosis<-c(a,b,c)

hist(kurtosis,freq=FALSE)
