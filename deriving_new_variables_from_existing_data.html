Deriving new variables from existing data

We have a data set results that has five variables:

Variable Type Level
class numeric nominal
gender numeric (binary) nominal
maths numeric ratio
english numeric ratio
history numeric ratio

The variables maths,english,history are examination scores out of 100. The class variable simply tells us which of three teachers taught the pupils. The gender variable is a binary female/male distinction.

We want to create and add a variable avxm which is the average of the three examination scores. We run the following code:

avxm<-sum(maths,english,history)/3

and add it to the dataframe with

cbind(results,avxm)
